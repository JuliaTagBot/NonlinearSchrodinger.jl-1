NLSS.Simulation.T₁ʰ — Method
T₁ʰ(ψ, ω, dx, F)

Compute ψ', i.e. ψ advanced a step dx forward using a symplectic second order integrator. ψ' is defined on an FFT grid with frequencies ω using an FFT plan F. Do not call this explicitly and use solve! instead.

See also: solve!

function plot_IoM(sim::Sim, x_res::Int64 = 500)

Plots the integrals of motion for a Sim object sim with a resolution x_res points in the x-direction. Produces plots of the energy, kinetic energy, potential energy, energy error, particle number and momentum
